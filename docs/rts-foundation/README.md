# RevTechStudio Foundation Plugin

## プラグイン要素一覧

### Convention Skill

| 要素名 | 概要 | 完了 |
| ------ | ---- | --- |
| coding-conventions | .NET/C#のコーディング規則（C# 12/13/14の最新機能優先使用、アンダースコアプレフィックス禁止、中括弧省略禁止） | 完 |
| development-principles | 開発における基本原則（曖昧さ禁止、矛盾検出、推察禁止） | 完 |
| documentation-standards | 全ドキュメントで共通して遵守するルールやガイドライン（構造の標準化、見出しレベル、用語統一、バージョン管理の記述方法） | 完 |
| interaction-guidelines | ユーザーとの対話パターンを定義（質問生成、曖昧さ解消、合意形成、フィードバック収集、ユーザー負担軽減） | 完 |
| issue-management-guidelines | Issue投稿・テンプレート等 | 完 |
| security-guidelines | 全フェーズで考慮すべきセキュリティ原則（認証・認可、機密情報取り扱い、セキュアコーディング、脆弱性チェックリスト） | 完 |
| version-control-guidelines | Git運用のガイドライン（ブランチ戦略、コミットメッセージ、プルリクエスト、マージ方針） | 完 |

### Agent

| 要素名 | 概要 | 完了 |
| ------ | ---- | --- |
| version-control-agent | Git操作およびGitHub操作全般を管理するエージェント | 完 |
