# rts-plugins

RevTechStudioチームが使用するClaude Codeのプラグインマーケットプレイスです。

## 概要

`RevTechStudio Plugins`は、開発チームの生産性を向上させるためのプラグインを提供するマーケットプレイスプラットフォームです。Claude Codeと統合し、開発ワークフローをシームレスにサポートします。

## 含まれるプラグイン

### rts-foundation

RevTechStudioプロジェクトの基盤となるプラグインです。共通機能、ユーティリティ、標準設定を提供します。

### rts-plugin-generator

新しいプラグインを素早く生成するためのジェネレータープラグインです。テンプレートベースでプラグインの骨格を自動生成し、開発時間を短縮します。

### rts-spec-dev

仕様書駆動開発を支援するプラグインです。要件定義から設計、実装までのドキュメント管理をサポートします。
